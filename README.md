# Exchange Online Lab

📧 Microsoft Exchange Online Homelab
A practical lab demonstrating core skills in Microsoft Exchange Online. This lab focuses on user setup, basic email filtering, message tracing, shared mailboxes, groups, and targeted transport rules.

🧭 Lab Overview
This lab covers:
- User creation and licensing
- Shared mailbox configuration
- Group and distribution list management
- Basic inbound email filtering
- Message tracing
- Transport rules for blocking risky attachments

👤 1. User Setup
Tasks completed:
- Create a new Microsoft 365 user
- Assign the appropriate Exchange Online license
  
Skills demonstrated:
- User provisioning
- License assignment

📬 2. Shared Mailbox Configuration
Tasks completed:
- Create a shared mailbox
- Add members to the shared mailbox
- Assign delegation permissions:
  - Send As
  - Full Access
  
Skills demonstrated:
- Shared mailbox administration
- Delegated access configuration

👥 3. Group & Distribution List Management
Tasks completed:
- Create a Microsoft 365 Group
- Create a Distribution List
- Assign members 
  
Skills demonstrated:
- Group and distribution list management
-	Membership configuration

🔍 4. Message Trace
Tasks completed:
- Perform message trace searches to confirm whether emails were blocked or delivered
- Validate sender behaviour (should or should not have been blocked)
  
Skills demonstrated:
- Message trace analysis
- Troubleshooting email delivery

🛡️ 5. Transport Rule: Block Executable Attachments
Tasks completed:
- Create a transport rule to block external emails containing executable attachments
- Configure the rule to forward blocked messages to a designated user for review
  
Skills demonstrated:
- Mail flow rule configuration
- Attachment filtering
- Targeted email blocking using transport rules
- Basic inbound email protection

🎯 Skills Demonstrated
- Exchange Online administration
- User provisioning and licensing
- Basic email filtering
- Message trace analysis
- Shared mailbox configuration
- Group and distribution list administration
- Transport rule creation
- Understanding of Exchange Online fundamentals
